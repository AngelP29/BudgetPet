# [BudgetPet](https://monetee.xyz/)

- BudgetPet allows you to track your expenses, set monthly budgets along saving goals, and ask your friendly manatee any financial questions you may have.
- Use our budget app in an interactive way to keep your finances in check anywhere you want. Mobile version available!

Technologies used
- MERN Stack
- Flutter/Dart

Features
- Level up your pet by staying within your budget and asking questions
- Reset password via email
- Flutter IOS mobile based component allows you to access your account anywhere
